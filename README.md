# Akshit Portfolio

Modern Next.js portfolio website with a polished UI, resume-driven sections, and a backend contact API.

## What to edit

Replace the placeholder links and contact details in `src/data/portfolio.ts` with your real profiles.

## Run locally

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Open `http://localhost:3000`

## Contact / Links

Edit the placeholder links and contact details in `src/data/portfolio.ts` to surface your GitHub, LinkedIn, email, LeetCode, and project URLs. Add any contact endpoint or form integration you prefer — this scaffold no longer includes a built-in contact route.
